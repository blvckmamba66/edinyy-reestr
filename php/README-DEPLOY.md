# Развёртывание на хостинге reg.ru (PHP + MySQL)

Это PHP-версия сайта «Единый реестр» для обычного виртуального хостинга (reg.ru, ISPmanager).
Данные хранятся в MySQL и не теряются. Сайт всегда онлайн.

## Что понадобится
- Хостинг reg.ru с PHP 7.4+ и MySQL (есть по умолчанию).
- Домен (можно привязать позже).

---

## Шаг 1. Создать базу данных MySQL
1. Панель reg.ru (ISPmanager) → раздел **«Базы данных»** → **Создать базу данных**.
2. Запиши: **имя базы**, **имя пользователя**, **пароль**. Хост обычно `localhost`.

## Шаг 2. Прописать данные базы в конфиге
Открой файл `includes/config.php` и заполни:
```php
define('DB_NAME', 'имя_базы');
define('DB_USER', 'пользователь');
define('DB_PASS', 'пароль');
```
Там же смени пароль администратора:
```php
define('ADMIN_PASSWORD', 'свой-надёжный-пароль');
```

## Шаг 3. Загрузить файлы на хостинг
1. Панель reg.ru → **«Менеджер файлов»** (или по FTP).
2. Загрузи **содержимое папки `php/`** (всё: `index.html`, `admin.html`, папки `api`, `includes`, `css` и т.д.)
   в корневую директорию сайта (обычно `.../data/www/твой-домен/` или `public_html`).

> Важно: загружать нужно **содержимое** папки `php/`, а не саму папку.

## Шаг 4. Проверить установку
Открой в браузере: `https://твой-домен/install.php`
- Страница проверит PHP, базу и **создаст таблицы**.
- Если всё зелёное — готово.
- **После проверки удали файл `install.php`** с хостинга.

## Шаг 5. Открыть сайт
- Сайт: `https://твой-домен/`
- Админка: `https://твой-домен/admin.html` (логин `admin`, пароль — какой задал в config.php)

---

## SMS-подтверждение
По умолчанию работает **демо-режим**: код показывается на экране (бесплатно).

Чтобы слать реальные SMS через SMS.ru — впиши в `includes/config.php`:
```php
define('SMSRU_API_ID', 'твой-api-id');
define('SMSRU_FROM', 'твой-домен');   // одобренное имя отправителя (или оставь пустым)
```

## Резервные копии
В ISPmanager есть раздел «Резервные копии» — настрой автокопирование базы, чтобы данные были в безопасности.

## Обновление пароля админа
Меняется в `includes/config.php` → `ADMIN_PASSWORD`.
