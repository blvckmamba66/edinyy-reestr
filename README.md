# Единый реестр граждан

Демонстрационный сайт на русском языке: регистрация пользователей по номеру телефона,
сбор персональных данных с согласием на обработку и админ-панель с экспортом в Excel.

## Возможности

- **Регистрация** по форме: Имя, Фамилия, Пол (М/Ж), Место рождения, Гражданство, Номер телефона.
- **Обязательная галочка** согласия на обработку персональных данных.
- **Вход** по номеру телефона (без кода подтверждения), личный кабинет с просмотром данных.
- **Админ-панель** (`/admin`): просмотр всех записей, поиск, редактирование данных,
  комментарии администратора, удаление пользователей, статистика.
- **Выгрузка всех данных в Excel** (`.xlsx`) одной кнопкой.

## Технологии

- Node.js + Express
- Встроенный `node:sqlite` (файл `data.db` создаётся автоматически) — нативных зависимостей нет
- ExcelJS для экспорта
- HTML / CSS / ванильный JavaScript на фронтенде

## Запуск

```powershell
npm install
npm start
```

Затем откройте:

- Сайт: <http://localhost:3000>
- Админ-панель: <http://localhost:3000/admin>

## Доступ администратора

По умолчанию: **логин** `admin`, **пароль** `admin123`.

Измените их через переменные окружения перед запуском:

```powershell
$env:ADMIN_LOGIN = "myadmin"
$env:ADMIN_PASSWORD = "сложный-пароль"
$env:SESSION_SECRET = "случайная-строка"
npm start
```

## Примечания по безопасности

Это учебно-демонстрационный проект. Для реальной эксплуатации необходимо:
подтверждение телефона по SMS-коду, хеширование/защита учётных данных администратора,
HTTPS, хранилище сессий вне памяти процесса и соответствие требованиям 152-ФЗ
«О персональных данных».
